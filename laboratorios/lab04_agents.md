# Laboratório 04: Construindo Agentes

## Objetivo
Criar o primeiro agente.

## Arquitetura
A interação básica de um agente autônomo segue este padrão:
`Usuário` ---> `Agent` ---> `Tool` ---> `Resposta`

## Exercício
**Objetivo:** Gerar um README automaticamente.

**Plano:**
1. Ler projeto
2. Identificar módulos
3. Criar documentação
4. Salvar arquivo

## Exercício Avançado
Criar um **ResearchAgent**.
Capaz de:
- consultar documentos;
- resumir artigos;
- gerar referências.
