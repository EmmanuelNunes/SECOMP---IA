# Laboratório 02: Prompt Engineering

## Objetivo
Construir prompts profissionais.

## Estrutura Base
Para prompts complexos, seguimos a seguinte estrutura:
- **ROLE** (Papel do Assistente)
- **TASK** (Tarefa a ser executada)
- **CONTEXT** (Contexto do problema)
- **CONSTRAINTS** (Restrições ou regras)
- **OUTPUT FORMAT** (Formato de saída desejado)

## Exercício
Criar um prompt para um **Analista de Currículos**.

**Exemplo:**
> Você é um recrutador sênior.
> Analise o currículo fornecido.
> Identifique:
> - pontos fortes
> - pontos fracos
> - sugestões
> 
> Responda em Markdown.

## Desafio
Criar versões deste mesmo prompt alterando a persona para:
1. Analista de RH
2. Especialista em Segurança da Informação
3. Cientista de Dados
