# Laboratório 03: MCP na Prática

## Objetivo
Compreender a arquitetura MCP (Model Context Protocol).

## Exercício Conceitual
Identifique a qual grupo (Tool, Resource ou Prompt) o seguinte provedor pertence:
- **GitHub**
- *Resposta:* Tool

## Exercício
Mapeie as seguintes integrações aos componentes corretos do MCP:
- Filesystem
- Database
- Web Search

## Diagrama
A arquitetura de conexão funciona através deste fluxo:
`Agent` ---> `MCP` ---> `GitHub` / `Filesystem` / `Database`
